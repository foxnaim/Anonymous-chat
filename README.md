# Anonymous Chat Frontend

Стартовый шаблон на Next.js 14 (App Router) с настроенными:

- TypeScript и абсолютными импортами `@/*`
- Tailwind CSS + кастомная цветовая схема
- Redux Toolkit (пример переключения темы)
- TanStack Query + Devtools
- next-seo, `robots.ts`, `sitemap.ts`
- Framer Motion, React Icons и Headless UI (пример переключателя темы)

## Getting started

```bash
yarn install
yarn dev
```

> Репозиторий настроен под Yarn Classic (1.x). Если нет глобально, установите: `npm install -g yarn`.

Дев-сервер доступен на http://localhost:3000.
